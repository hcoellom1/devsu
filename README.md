# Pasos para el despliegue

1. Dentro de este repositorio (raíz del proyecto - clientes) se encuentran los archivos DockerFile y docker-compose con las respectivas imágenes
2. Se debe ubicar dentro de la raíz del proyecto y levantar las imágenes del docker
3. Adicional, en este mismo repositorio se encuentra el archivo .json con la colección de endpoints de postman de todas las funcionalidades
4. También se encuentra el archivo BaseDatos.sql que contiene el script de creación de la base de datos
